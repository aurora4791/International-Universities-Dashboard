# Global Opportunities Universities Dashboard

A visually engaging, interactive dashboard to help students explore the best international universities for AI and Entrepreneurship.

## 🚀 Features

- Region and tuition filters
- Colorful design and responsive cards
- Real social media icons/links
- Expand/collapse details on each university card

## 📦 How to Use

1. **Open `index.html`** in your browser.
2. **Edit/Add university cards** in the HTML for new entries.
3. **Modify styles** in `dashboard.css` for your own color and flair.
4. **Deploy on GitHub Pages** (see below).

## 🌐 Deploying to GitHub Pages

1. Fork or clone this repo.
2. Push all files to a GitHub repo.
3. Go to Settings > Pages > Source.  
   Select `main branch` and `/root` as folder.
4. Visit the generated GitHub Pages URL!
